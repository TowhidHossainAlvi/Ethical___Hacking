======== elpscrk ==========

---- installation ---------
 
- git clone https://github.com/D4Vinci/elpscrk
- cd elpscrk
- pip3 install -r requirements.txt
- ls 
- ls -la  (permission checking)
- chmod +x elpscrk.py  (giving permission)
- python3 elpscrk.py

- python3 elpscrk.py --help
- python3 elpscrk.py -c -l 2 --min 5 --max 6
